# bevy_joycons

A WIP bevy plugin for handling Nintendo Switch Joycon controllers, based on
https://github.com/Yamakaky/joy.

Currently supports bevy 0.9.

## External dependencies

(copied from the inner joycon crate's readme)

On Linux, you'll need `libusb`, `libbluetooth` and `libudev`. On Ubuntu, you can
install these by running:

```sh
sudo apt-get install libusb-1.0-0-dev libbluetooth-dev libudev-dev
```
